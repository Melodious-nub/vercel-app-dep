Long-term leave

Long-term leave feature is meant for long-term absences like maternity leave, where certain policies need to be put on "pause" so they don't accrue while an employee is away. If you want to add occasional days when an employee will be away, it's best to create a new time off policy called "Unpaid leave" & let your employees use that one instead.
